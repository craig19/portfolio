Features Used



Bootstrap and extra misc. features used



  • Bootstrap carousel: used this feature to implement a slideshow for cycling through images of my 4th year project. Provides user with visual detail of my project. See index.html#projects


  • Bootstrap collapse: used this feature to allow user to control how much data they see. Prevents too much information being displayed to the user at any one time. See index.html#education and #projects


  • Bootstrap tabs/pills: used this feature to separate content into different panes allowing the user to control which pane & information they wish to see. See index.html#education


  • Bootstrap modal: implemented Bootstrap modal to allow user to send a contact message if they wish. See index.html#about


  • Search bar: implemented search bar which would allow users to find what they are looking for as fast as possible.


  • Site Map Links in Footer: implemented site map which would allow users to navigate to what they are looking for as fast as possible.


  • Font-awesome library: used this libraries' icons throughout the site for better visualizations. Also added various interactivity to icons for a better user experience.



  • Working icon/picture links to allow user to get as much information as possible if they wish, depending on which icon they press. Icons such as GitHub, LinkedIn, StackOverflow all open up new windows for users. TUDublin picture also allows user to navigate to TUDublin site if they wish.


  • Working CV download. Users can instantly download and view a full version of my CV if they wish. See index.html#about


  • Google Maps API: implemented Google Maps into contact section to add to a better visual feel for the user, also provides users with additional location information. See index.html#contact


  • Implemented percentage bars and percentage change to grade on hover adding extra interacticity to the website and providing the user with additional information. See index.html#education


  • Sticky navbar with scrollspy: to give a more modern feel for users and to allow better website navigation, I used scrollspy to automatically update which nav link is active, depending on what section the user scrolls to/is viewing. Also does not instantly jump to section on-click.


  • All features implemented using Bootstrap grid columns & rows in the correct way, ensuring that content is usable & legible on mobile, tablet and desktop devices. (I used chromes developer features to make sure)

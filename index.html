Fourth Year Major Project



Project Overview

For my fourth year college project I am working on a live project with CSinc.ie, which is a team made up of lecturers from the TU Dublin, Tallaght campus. This team of lecturers takes bookings from schools and organisations all over Ireland, providing computer science outreach camps to students and personal development camps to teachers. The team also carries out longitudinal surveys, which aim to help develop the outreach camps moving forward and in turn, allow lecturers to reach and inform as many students as possible of what computer science really is; addressing sterotypes and incorrect perceptions of the subject.

My role is to design, develop, implement and test a full-stack booking system for the organisation, which can handle thousands of bookings. The system will allow lecturers to manage bookings and available camp dates. The system will also facilitate the upload of thousands of scanned hand-written surveys: lecturers collect student and teacher surveys at every school camp which need to be entered into a database for research and data analysis purposes. Lecturers will be able to upload a .PDF for each school containing all of the schools surveys. The system will then scan over every survey in the .PDF and use two extraction methods to extract the data for entry into the database.

The first data extraction method involves obtaining and parsing survey answers which are hand-written. I have leveraged the Azure Optical Character Recognition Read API to parse hand-written text into ASCII characters which will then be available for use in the system. The second extraction method involves obtaining hand-marked checkbox data. I have came up with solid logic and algorithms to scan every checkbox pixel by pixel, measuring the pixel density of each checkbox against an un-marked base pixel density value, to determine if the checkbox has been marked or not.

This project has provided me with valuable experience in the following technologies: C#, ASP.NET MVC, Bootstrap, Azure DevOps CI/CD, automated testing, AWS Elastic Beanstalk & RDS, Azure Cognitive Services API, Google Vision OCR API, AWS Textract OCR API (I tested each of these 3 OCR APIs for accuracy with hand-written survey answers; Azures Read API provided the highest level of accuracy).
